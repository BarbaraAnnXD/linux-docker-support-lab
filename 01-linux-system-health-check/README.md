# Lab 01: Linux System Health Check

## Objective

Practice basic Linux system health checks used in technical support and cybersecurity support roles.

## Scenario

A user reports that a Linux system is running slowly or behaving unexpectedly. The goal is to collect basic system information, check resource usage, review running processes, and document findings clearly.

## Commands Practiced

```bash
whoami
hostname
uname -a
uptime
df -h
free -h
ps aux
top
ip addr
ping -c 4 google.com
