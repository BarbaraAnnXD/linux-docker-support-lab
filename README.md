# Linux Docker Support Lab

This repository documents hands-on Linux and Docker troubleshooting labs focused on technical support, system behavior, service health, logs, ports, containers, and clear support-style documentation.

## Purpose

The purpose of this lab portfolio is to build practical systems troubleshooting skills for cybersecurity support, cloud support, and security systems engineering roles.

## Focus Areas

- Linux command-line troubleshooting
- File permissions and process checks
- Network connectivity testing
- Docker container basics
- Logs and error analysis
- Port and service troubleshooting
- Technical support documentation
- Ticket-style investigation notes
